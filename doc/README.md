# EGE 图形库文档

这里包含了 EGE (Easy Graphics Engine) 图形库的完整文档，使用 Doxygen 从源代码自动生成。

## 文档结构

- **html/**: HTML 格式的文档（主要使用）
- **latex/**: LaTeX 格式的文档（可生成 PDF）
- **Doxyfile**: Doxygen 配置文件

## 生成文档

要重新生成文档，请在此目录下运行：

```bash
doxygen Doxyfile
```

## 访问文档

- 打开 `html/index.html` 查看完整文档
- 或者访问项目根目录的 `index.html` 获取导航页面

## 文档分类

文档已按功能自动分类：

- **图形绘制**: 基本图形绘制函数
- **图像管理**: 图像加载、保存和处理
- **输入处理**: 键盘和鼠标输入处理
- **文本输出**: 文本显示和字体设置
- **变换操作**: 图形变换和矩阵操作
- **路径管理**: 复杂图形路径操作
- **控制台操作**: 控制台相关功能
- **时间控制**: 时间和延时功能
