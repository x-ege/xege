EGE图形库的核心图像绘制功能 

putimage系列函数提供了丰富的图像绘制和处理能力，包括：
\begin{DoxyItemize}
\item 基础图像绘制：putimage() 系列重载函数
\item 透明效果：putimage\+\_\+transparent(), \doxylink{group__putimage__functions_gaddd5501a4b12fe9515e21dcbb042fa70}{putimage\+\_\+alphablend()}, \doxylink{group__putimage__functions_gabfa951d87707dcfd74025fb6d345f97e}{putimage\+\_\+withalpha()}
\item 混合效果：putimage\+\_\+alphatransparent(), \doxylink{group__putimage__functions_ga64ab75c02417b7bf4680172ca791a801}{putimage\+\_\+alphafilter()}
\item 变换效果：putimage\+\_\+rotate(), \doxylink{group__putimage__functions_gac9e5123fe64c4acbaf28b8d38dd33a88}{putimage\+\_\+rotatezoom()}, \doxylink{group__putimage__functions_ga6fc0966df910129130301cb4c051da47}{putimage\+\_\+rotatetransparent()}
\end{DoxyItemize}

这些函数支持多种绘制模式：
\begin{DoxyItemize}
\item 绘制到屏幕（img\+Dest = NULL）或另一个图像
\item 支持区域裁剪、拉伸缩放、旋转变换
\item 支持多种透明和混合模式
\item 支持光栅操作码（\+SRCCOPY, SRCAND, SRCPAINT等）
\item 支持平滑处理（抗锯齿） 
\end{DoxyItemize}
