\doxysubsection{详细描述}
音乐播放类 

\doxylink{classege_1_1MUSIC}{MUSIC} 类提供了基于 Windows Media Control Interface (MCI) 的音乐播放功能， 支持播放 WAV、\+MP3、\+MIDI 等多种音频格式。

\begin{DoxyNote}{注解}
该类基于 Windows MCI 实现，仅支持 Windows 平台 

支持的音频格式包括：\+WAV, MP3, MIDI 等 
\end{DoxyNote}
