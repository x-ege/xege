\doxysubsubsubsection{详细描述}
EGE图形库的图像创建、加载、保存和处理功能 

图像管理模块提供了完整的图像操作能力，包括：
\begin{DoxyItemize}
\item 图像缓冲区操作：getbuffer() 获取像素缓冲区
\item 图像尺寸调整：resize(), \doxylink{group__image__management_ga242ea4c710974781ce9475afa4d1b8be}{resize\+\_\+f()} 调整图像大小
\item 图像获取：getimage() 系列函数从不同源获取图像数据
\item 图像保存：saveimage(), \doxylink{group__putimage__functions_gaa44071ee7e219391718448560de01cbc}{savepng()}, \doxylink{group__putimage__functions_ga213160e5a0b8998a632c678f21e5f14f}{savebmp()} 保存图像到文件
\end{DoxyItemize}

支持的图像格式：\+PNG, BMP, JPG, GIF, EMF, WMF, ICO 支持从窗口、文件、资源、其他\+IMAGE对象获取图像
