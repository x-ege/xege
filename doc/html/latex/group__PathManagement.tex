创建一个新的图像对象（1x1像素） 

\begin{DoxyReturn}{返回}
返回新创建的图像对象指针，失败时返回\+NULL
\end{DoxyReturn}
创建一个大小为1x1像素的图像对象，背景色为黑色。 创建后的图像需要使用delimage()函数销毁以防止内存泄漏。

\begin{DoxyNote}{注解}

\end{DoxyNote}
\begin{DoxyWarning}{警告}

\end{DoxyWarning}
