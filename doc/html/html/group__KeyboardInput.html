int EGEAPI ege::keystate ( int  key)

判断键盘或鼠标上的某按键是否处于按下状态

参数
key要检测的按键代码，参见key_code_e枚举
返回
非零值表示按键按下，0表示未按下，-1表示参数错误
注解
实时检测按键状态，不消耗消息队列；支持键盘和鼠标按键
参见
key_code_e
